# Customer Feedback Automation System (n8n + AI)

##  Overview
This project is an AI-powered customer feedback system that collects, classifies, and processes user feedback in real time.

##  Tech Stack
- n8n (Workflow Automation)
- Google Gemini AI
- Airtable (Database)
- Slack / Gmail (Notifications)

##  Features
- Collect feedback via live form
- AI-based classification:
  - Complaint
  - Compliment
  - Suggestion
- Automated data storage in Airtable
- Real-time notifications
